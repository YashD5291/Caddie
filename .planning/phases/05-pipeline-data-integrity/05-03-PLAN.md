    1. Add static method to AudioFileManager:
       ```swift
       /// Removes orphaned caddie_mono_* temp files from previous sessions.
       /// Called on app startup to reclaim disk space from crashed transcriptions.
       static func cleanupOrphanedTempFiles() {
           let tempDir = FileManager.default.temporaryDirectory
           let fm = FileManager.default
           guard let contents = try? fm.contentsOfDirectory(
               at: tempDir,
               includingPropertiesForKeys: nil
           ) else { return }

           let orphans = contents.filter { $0.lastPathComponent.hasPrefix("caddie_mono_") }
           for orphan in orphans {
               try? fm.removeItem(at: orphan)
           }

           if !orphans.isEmpty {
               Logger(subsystem: "com.caddie.app", category: "AudioFileManager")
                   .info("Cleaned up \(orphans.count) orphaned temp file(s)")
           }
       }
       ```

    - State machine transitions recording -> error via .recordingFailed event
    - AppState's onStateChange callback receives .error state

    Verify this chain is complete by tracing:
    1. RecordingCoordinator.executeStartRecording() catch block calls handle(.recordingFailed)
    2. RecordingState.reduce handles (.recording, .recordingFailed) -> (.error, .notifyError)

    4. AppState onStateChange receives .error and sets status = .idle

    The one gap: AppState sets status = .idle on error but does NOT surface the error message
