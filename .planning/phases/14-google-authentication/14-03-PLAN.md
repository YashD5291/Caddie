    Add the `CFBundleURLTypes` key inside the root `<dict>`, before the closing `</dict>`:
    ```xml
    <key>CFBundleURLTypes</key>
    <array>
        <dict>
            <key>CFBundleURLSchemes</key>
            <array>
                <string>com.googleusercontent.apps.REPLACE_WITH_GOOGLE_CLIENT_ID</string>

        </dict>
    </array>
    ```

    This registers the custom URI scheme so macOS routes the Google OAuth redirect back to Caddie (per Pitfall 5 in research). The REPLACE_WITH placeholder matches GoogleOAuthConfig.callbackScheme -- user replaces both at the same time when they create their Google Cloud Console client ID.


    Add a mirror property for observable UI binding (since GoogleAuthManager is an actor, SwiftUI cannot observe it directly):
    ```swift

    ```

    In `initialize()`, BEFORE the existing database setup, call session restoration:
    ```swift
    // Restore Google auth session from Keychain (AUTH-01)
    await authManager.restoreSession()
    googleAuthState = await authManager.state
    ```

    Add helper methods for UI actions (matching the existing startManualRecording/stopManualRecording pattern):
    ```swift
    func signInToGoogle(window: NSWindow) {
        Task {
            do {
                try await authManager.signIn(presenting: window)
                googleAuthState = await authManager.state
            } catch {
                CaddieLogger.auth.error("Google sign-in failed: \(error.localizedDescription)")
                googleAuthState = await authManager.state
            }
        }
    }

    func signOutFromGoogle() {
        Task {
            await authManager.signOut()
            googleAuthState = await authManager.state
        }
    }
    ```

    **1. Create GoogleAccountSection** (Sources/UI/Settings/GoogleAccountSection.swift):

    ```swift
    import SwiftUI

    struct GoogleAccountSection: View {
        @Environment(AppState.self) private var appState

        var body: some View {
            Section("Google Account") {
                switch appState.googleAuthState {
                case .signedOut:
                    signedOutView
                case .signingIn:
                    signingInView
                case .signedIn(let email):
                    signedInView(email: email)
                case .error(let message):
                    errorView(message: message)
                }
            }
        }

        // MARK: - State Views

        private var signedOutView: some View {
            VStack(alignment: .leading, spacing: 8) {
                Text("Sign in to sync your Google Calendar and auto-record meetings.")
                    .font(.caption)
                    .foregroundStyle(.secondary)

                Button("Sign in with Google") {
                    guard let window = NSApplication.shared.keyWindow else { return }
                    appState.signInToGoogle(window: window)
                }
            }
        }

        private var signingInView: some View {
            HStack {
                ProgressView()
                    .controlSize(.small)
                Text("Signing in...")
                    .foregroundStyle(.secondary)
            }
        }

        private func signedInView(email: String) -> some View {
            VStack(alignment: .leading, spacing: 8) {
                HStack {
                    Image(systemName: "checkmark.circle.fill")
                        .foregroundStyle(.green)
                    Text("Connected as \(email)")
                }

                Button("Sign Out") {
                    appState.signOutFromGoogle()
                }
            }
        }

        private func errorView(message: String) -> some View {
            VStack(alignment: .leading, spacing: 8) {
                Label(message, systemImage: "exclamationmark.triangle")
                    .font(.caption)
                    .foregroundStyle(.orange)

                Button("Try Again") {
                    guard let window = NSApplication.shared.keyWindow else { return }
                    appState.signInToGoogle(window: window)
                }
            }
        }
    }
    ```

    Key UI decisions:
    - Shows contextual content based on authState (signedOut/signingIn/signedIn/error)
    - "Connected as email" with green checkmark when signed in (per roadmap success criteria)
    - Sign Out button available when signed in (AUTH-04)
    - Error state shows message + retry button
    - Uses NSApplication.shared.keyWindow for presentation context (per Pitfall 4)
    - Caption text explains WHY to sign in ("sync your Google Calendar")
